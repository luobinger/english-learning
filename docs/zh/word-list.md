## 学习建议

::: tip 高效记忆单词的方法
1. **结合代码场景**：在实际开发中遇到生词时，记录下来
2. **多次重复**：利用碎片时间反复复习
3. **实际使用**：尝试用新学的单词写注释、文档
4. **词根词缀**：学习常见的词根词缀，举一反三
:::

::: warning 注意
单词的记忆需要时间和耐心，不要期望一次记住所有单词。每天坚持学习一点点，积少成多。
:::
