English vocabulary lists specifically curated for programmers, covering common technical terms in daily development.

## Common (General Technical Terms)

These are the most basic and commonly used English words in software development.

| Word | Meaning | Example |
|------|---------|---------|
| abstraction | 抽象 | The abstraction layer hides the complexity of the implementation. |
| algorithm | 算法 | We need to optimize this algorithm for better performance. |
| architecture | 架构 | The microservice architecture allows for better scalability. |
| asynchronous | 异步的 | Asynchronous programming can improve application responsiveness. |
| atomic | 原子的 | Database operations should be atomic to ensure data consistency. |
| benchmark | 基准测试 | We need to run benchmarks to measure the performance improvement. |
| binary | 二进制 | The binary search algorithm has O(log n) time complexity. |
| cache | 缓存 | We should cache the frequently accessed data to reduce database load. |
| callback | 回调 | The callback function will be executed when the async operation completes. |
| compile | 编译 | The compiler will check for syntax errors during compilation. |

## Learning Tips

::: tip Efficient Memorization Methods
1. **Combine with code scenarios**: Record new words encountered during actual development
2. **Multiple repetitions**: Use fragmented time to repeatedly review
3. **Actually use**: Try to use new words in comments and documentation
4. **Roots and affixes**: Learn common roots and affixes, extrapolate
:::

::: warning Note
Memorizing words requires time and patience. Don't expect to remember all words at once. Learn a little every day, accumulate over time.
:::
